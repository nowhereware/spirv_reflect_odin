# spirv_reflect_odin
Odin bindings for SPIRV-Reflect

Current matching commit: https://github.com/KhronosGroup/SPIRV-Reflect/commit/10b4f09a24d7ac1603071e767c089551dc6a3949

## NOTE:
These bindings were written on a Windows-based system, so other platform support hasn't been fully tested yet. Additionally, these bindings directly reference the library locally although I do not have access to the proper lib to place in this folder. If you have any issues, please submit a PR!
